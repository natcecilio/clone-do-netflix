# Netflix Clone
![enter image description here](https://upload.wikimedia.org/wikipedia/commons/0/0f/Logo_Netflix.png)

-   Demo:  [https://clone-do-netflix-git-master.natcecilio.vercel.app/](https://clone-do-netflix-git-master.natcecilio.vercel.app/)

This project is a simplified front end clone of Netflix. It was created with React JS. 
It uses  [The MovieDB Api](https://www.themoviedb.org/documentation/api)  to search for movies and display details.
